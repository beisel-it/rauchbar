# Site

Marketing-Site, Mitglieder-Homepage und zeitverzoegerte Deal-Oeffentlichkeit.

## UX Scope

- Homepage with clear member-value proposition
- delayed public deal archive with 24h exclusivity messaging
- signup and preference onboarding
- member home for delivery status and quick settings

## Required Navigation

- Deals entdecken
- So funktioniert's
- Mitglied werden
- Login

Member area:

- Fuer dich
- Alerts
- Praeferenzen
- Konto
